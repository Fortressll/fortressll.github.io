Leetcode-100.相同的树(Same Tree)

# Sep 25, 2019

采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

100.相同的树(Same Tree)

采用递归的方法，首先判断两根节点是否为空，若均为空返回true，有一个为空返回false，然后判断两根节点是否相等，若相等则对他们的左节点和右节点做相同的递归操作，否则返回false

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public boolean isSameTree(TreeNode p, TreeNode q) {
        if (p == null & q == null)
            return true;
        if (p == null || q == null)
            return false;
        if (p.val != q.val)
            return false;
        boolean result = isSameTree(p.left, q.left) && isSameTree(p.right, q.right);

        return result;
    }
}

执行用时：0ms 内存消耗：34.2MB
