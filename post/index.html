Leetcode-100.相同的树(Same Tree)

Sep 25, 2019

采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

100.相同的树(Same Tree)

采用递归的方法，首先判断两根节点是否为空，若均为空返回true，有一个为空返回false，然后判断两根节点是否相等，若相等则对他们的左节点和右节点做相同的递归操作，否则返回false

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public boolean isSameTree(TreeNode p, TreeNode q) {
        if (p == null & q == null)
            return true;
        if (p == null || q == null)
            return false;
        if (p.val != q.val)
            return false;
        boolean result = isSameTree(p.left, q.left) && isSameTree(p.right, q.right);

        return result;
    }
}

执行用时：0ms 内存消耗：34.2MB

Leetcode-202.快乐数(Happy Number)

Sep 24, 2019

采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

202.快乐数(Happy Number)

题目要求判断给定数是否是快乐数，利用HashSet，如果集合中不存在每位运算的平方和，则加入，否则集合中之前已经出现同样的数，说明再循环计算下去也一定不会是快乐数

class Solution {
    Set<Integer> set = new HashSet<>();
    public boolean isHappy(int n) {
        while (n != 1) {
            n = happySquare(n);
            if (set.contains(n)) {
                return false;
            } else {
                set.add(n);
            }
        }
        return true;
    }
    private int happySquare(int n) {
        int result = 0;
        do {
            int last = n % 10;
            result += last * last;
            n /= 10;
        } while (n > 0);
        return result;
    }
}

执行用时：3ms 内存消耗：33.6MB

Leetcode-169.求众数(Majority Element)

Sep 21, 2019

采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

169.求众数(Majority Element)

解法一

排序已知数组，对应下标length/2的位置恰好是众数

class Solution {
     public int majorityElement(int[] nums) {
        Arrays.sort(nums);
        return nums[nums.length / 2];
    }
}

执行用时：2ms 内存消耗：41.5MB

解法二

穷举，超级费时间

class Solution {
    public int majorityElement(int[] nums) {
        int half = nums.length/2;

        for (int num : nums) {
            int count = 0;
            for (int item : nums) {
                if (item == num) {
                    count += 1;
                }
            }
            if (count > half) {
                return num;
            }
        }
        return -1;
    }
}

执行用时：1725ms 内存消耗：40.7MB
