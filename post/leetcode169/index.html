Leetcode-169.求众数(Majority Element)

# Sep 21, 2019

采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

169.求众数(Majority Element)

解法一

排序已知数组，对应下标length/2的位置恰好是众数

class Solution {
     public int majorityElement(int[] nums) {
        Arrays.sort(nums);
        return nums[nums.length / 2];
    }
}

执行用时：2ms 内存消耗：41.5MB

解法二

穷举，超级费时间

class Solution {
    public int majorityElement(int[] nums) {
        int half = nums.length/2;

        for (int num : nums) {
            int count = 0;
            for (int item : nums) {
                if (item == num) {
                    count += 1;
                }
            }
            if (count > half) {
                return num;
            }
        }
        return -1;
    }
}

执行用时：1725ms 内存消耗：40.7MB
