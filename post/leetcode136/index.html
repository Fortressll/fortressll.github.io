136.只出现一次的数字(Single Number)

对已知数组排序，然后遍历数组，寻找只出现一次的元素

class Solution {
    public int singleNumber(int[] nums) {
        Arrays.sort(nums);

        for (int i = 0; i < nums.length - 1; i+=2) {
            if (nums[i] != nums[i+1])
                return nums[i];
        }
        return nums[nums.length-1];
    }
}

执行用时：5ms 内存消耗：38.1MB
