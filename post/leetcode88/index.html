Leetcode-88.合并两个有序数组(Merge Sorted Array)

# Sep 21, 2019

  • 采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

88.合并两个有序数组(Merge Sorted Array)

把nums2里的元素直接接在nums1后面然后排序

class Solution {
    public void merge(int[] nums1, int m, int[] nums2, int n) {
        int index = 0;
        for (int i = m; i < m + n; i++) {
            nums1[i] = nums2[index++];
        }
        Arrays.sort(nums1);
    }
}

执行用时：1ms 内存消耗：36.1MB
