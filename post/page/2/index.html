Leetcode-136.只出现一次的数字(Single Number)

Sep 21, 2019

采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

136.只出现一次的数字(Single Number)

对已知数组排序，然后遍历数组，寻找只出现一次的元素

class Solution {
    public int singleNumber(int[] nums) {
        Arrays.sort(nums);

        for (int i = 0; i < nums.length - 1; i+=2) {
            if (nums[i] != nums[i+1])
                return nums[i];
        }
        return nums[nums.length-1];
    }
}

执行用时：5ms 内存消耗：38.1MB

Leetcode-88.合并两个有序数组(Merge Sorted Array)

  • 采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

88.合并两个有序数组(Merge Sorted Array)

把nums2里的元素直接接在nums1后面然后排序

class Solution {
    public void merge(int[] nums1, int m, int[] nums2, int n) {
        int index = 0;
        for (int i = m; i < m + n; i++) {
            nums1[i] = nums2[index++];
        }
        Arrays.sort(nums1);
    }
}

执行用时：1ms 内存消耗：36.1MB

Leetcode-83.删除排序链表中的重复元素(Remove Duplicates from Sorted List)

  • LeetCode国内官网：https://leetcode-cn.com

简单难度

83.删除排序链表中的重复元素(Remove Duplicates from Sorted List)

参考了某大佬思路，题目要求是删除重复元素，首先判断链表是否为空或者链表是否只剩一个元素，这种情况直接返回就可以了；接下来进行递归操作，判断头节点值是否等于头节点后next节点的值，相等表明该元素重复，返回head.next值即可

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode deleteDuplicates(ListNode head) {
        if (head == null || head.next == null)
            return head;
        head.next = deleteDuplicates(head.next);
        if (head.val == head.next.val)
            head = head.next;
        return head;
    }
}

执行用时：2ms 内存消耗：37.4MB
