Leetcode-7.整数反转(Reverse Integer)

# Sep 11, 2019

  • 采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

7.整数反转(Reverse Integer)

该题要求将目标值反转，比如123转为321，主要是数学问题

我们可以设置一个值last，从个位数开始保存，依次加入十位数、百位数等

class Solution {
    public int reverse(int x) {
        long last = 0;
        while (x != 0) {
            last = last * 10 + x % 10;
            x /= 10;
        } 
        if (last < Integer.MIN_VALUE || last > Integer.MAX_VALUE) {
                last = 0;
        }
        return (int)last;
    }
}

执行用时：3ms 内存消耗：34.1MB
