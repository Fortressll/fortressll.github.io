Leetcode-202.快乐数(Happy Number)

# Sep 24, 2019

采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

202.快乐数(Happy Number)

题目要求判断给定数是否是快乐数，利用HashSet，如果集合中不存在每位运算的平方和，则加入，否则集合中之前已经出现同样的数，说明再循环计算下去也一定不会是快乐数

class Solution {
    Set<Integer> set = new HashSet<>();
    public boolean isHappy(int n) {
        while (n != 1) {
            n = happySquare(n);
            if (set.contains(n)) {
                return false;
            } else {
                set.add(n);
            }
        }
        return true;
    }
    private int happySquare(int n) {
        int result = 0;
        do {
            int last = n % 10;
            result += last * last;
            n /= 10;
        } while (n > 0);
        return result;
    }
}

执行用时：3ms 内存消耗：33.6MB
