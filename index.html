Leetcode-169.求众数(Majority Element)

Sep 21, 2019

采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

169.求众数(Majority Element)

解法一

排序已知数组，对应下标length/2的位置恰好是众数

class Solution {
     public int majorityElement(int[] nums) {
        Arrays.sort(nums);
        return nums[nums.length / 2];
    }
}

执行用时：2ms 内存消耗：41.5MB

解法二

穷举，超级费时间

class Solution {
    public int majorityElement(int[] nums) {
        int half = nums.length/2;

        for (int num : nums) {
            int count = 0;
            for (int item : nums) {
                if (item == num) {
                    count += 1;
                }
            }
            if (count > half) {
                return num;
            }
        }
        return -1;
    }
}

执行用时：1725ms 内存消耗：40.7MB

Leetcode-136.只出现一次的数字(Single Number)

Sep 21, 2019

采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

136.只出现一次的数字(Single Number)

对已知数组排序，然后遍历数组，寻找只出现一次的元素

class Solution {
    public int singleNumber(int[] nums) {
        Arrays.sort(nums);

        for (int i = 0; i < nums.length - 1; i+=2) {
            if (nums[i] != nums[i+1])
                return nums[i];
        }
        return nums[nums.length-1];
    }
}

执行用时：5ms 内存消耗：38.1MB

Leetcode-88.合并两个有序数组(Merge Sorted Array)

  • 采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

88.合并两个有序数组(Merge Sorted Array)

把nums2里的元素直接接在nums1后面然后排序

class Solution {
    public void merge(int[] nums1, int m, int[] nums2, int n) {
        int index = 0;
        for (int i = m; i < m + n; i++) {
            nums1[i] = nums2[index++];
        }
        Arrays.sort(nums1);
    }
}

执行用时：1ms 内存消耗：36.1MB
