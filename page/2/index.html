Leetcode-21.合并两个有序链表(Merge Two Sorted Lists)

Sep 19, 2019

  • 采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

21.合并两个有序链表(Merge Two Sorted Lists)

这道题目是一个有关链表的题目中，注释中给出了ListNode的定义

整个合并过程可以看作是一个递归的过程，两个链表中头部较小的一个与剩下的元素进行递归合并

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
        if (l1 == null) {
            return l2;
        } else if (l2 == null) {
            return l1;
        } else if (l1.val < l2.val) {
            l1.next = mergeTwoLists(l1.next, l2);
            return l1;
        } else {
            l2.next = mergeTwoLists(l1, l2.next);
            return l2;
        }
    }
}

执行用时：1ms 内存消耗：36.6MB

Leetcode-217&&219.存在重复元素(Contains Duplicate)相关

Sep 18, 2019

采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

217.存在重复元素(Contains Duplicate)

解法一

如果我们直接遍历做这道题，时间复杂度会有O(n^2)，我们可以先对数组排个序就可以通过了

class Solution {
    public boolean containsDuplicate(int[] nums) {
        Arrays.sort(nums);
        for (int i = 0; i < nums.length - 1; i++) {
            if (nums[i] == nums[i+1])
                return true;
        }
        return false;
    }
}

执行用时：9ms 内存消耗：48MB

解法二

用了哈希表，在线性的时间复杂度就可以解决，记住这个点，以后能用就用

class Solution {
    public boolean containsDuplicate(int[] nums) {
        Set<Integer> set = new HashSet<>(nums.length);
        for (int x: nums) {
            if (set.contains(x))
                return true;
            set.add(x);
        }
        return false;
    }
}

执行用时：14ms 内存消耗：44.5MB

219.存在重复元素II(Contains Duplicate II)

用到了HashMap,因为有key和value，个人觉得还挺好理解~要注意的一点就是存HashMap的时候，并不是按照i:nums[i]存的，而是把nums[i]存为key，i存为value，因为自带的get方法可以通过传入一个key来获取value，如果这样存就可以很方便的获得它的i值从而进行索引值i，j的计算，来达到题目要求。

class Solution {
    public boolean containsNearbyDuplicate(int[] nums, int k) {
        Map<Integer, Integer> map = new HashMap<>();

        for (int i = 0; i < nums.length; i++) {
            if (!map.containsKey(nums[i])) {
                map.put(nums[i], i);
            } else {
                int j = map.get(nums[i]);
                if (Math.abs(i - j) <= k)
                    return true;
                map.put(nums[i], i);
            }

        }
        return false;
    }
}

执行用时：29ms 内存消耗：49.1MB

Leetcode-349.两个数组的交集(Intersection of Two Arrays)

Sep 17, 2019

采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

349.两个数组的交集(Intersection of Two Arrays)

解法一

题目要求两个数组的交集，习惯性遍历两个数组来做，但是时间复杂度就会很高了。先对两个数组排序，然后选择较短的那个数组长度定义新数组，比较两个数组，将相同且新数组中没有的元素添加至新数组。注意：因为新数组长度可能较长，最后可能会有默认值0，所以截取其实际长度(index)作为返回值

笔记：Java中的Arrays.copyOfRange()函数用来截取数组中的某一段，就像是Python中的切片操作

class Solution {
    public int[] intersection(int[] nums1, int[] nums2) {
        Arrays.sort(nums1);
        Arrays.sort(nums2);
        int len = nums1.length > nums2.length ? nums2.length : nums1.length;
        int[] result = new int[len];
        int index = 0;

        for (int s1: nums1) {
            for (int s2: nums2) {
                if (s1 == s2 && index == 0) {
                    result[index++] = s1;
                } else  if (s1 == s2 && s1 != result[index-1]) {
                    result[index++] = s1;
                }
            }
        }

        return Arrays.copyOfRange(result, 0, index);
    }
}

执行用时：18ms 内存消耗：38.3MB

解法二

用了HashSet,也就是集合，将给定的两个数组都转为集合后，就可以直接用集合的交集函数，再将集合转为数组即可达到题目要求

笔记：Java中的retainAll()函数用来求集合

import java.util.HashSet;

class Solution {
    public int[] intersection(int[] nums1, int[] nums2) {
        HashSet<Integer> set1 = new HashSet<Integer>();
        HashSet<Integer> set2 = new HashSet<Integer>();
        for (Integer s1: nums1)
            set1.add(s1);
        for (Integer s2: nums2)
            set2.add(s2);

        set1.retainAll(set2);

        int[] result = new int[set1.size()];
        int i = 0;
        for (int s1: set1)
            result[i++] = s1;

        return result;
    }
}

执行用时：9ms 内存消耗：37.4MB
