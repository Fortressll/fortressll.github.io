Leetcode-35.探索插入位置(Search Insert Position)

Sep 16, 2019

采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

35.探索插入位置(Search Insert Position)

额~这个题还是很简单的，都有点不好意思写了，遍历下就完事了。当然是没考虑内存的情况，击败不了多少用户

class Solution {
    public int searchInsert(int[] nums, int target) {
        int index = 0;
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] == target || nums[i] > target) {
                index = i;
                break;
            }
            else {
                index = i + 1;
            }
        }
        System.out.println(index);
        return index;
    }
}

执行用时：3ms 内存消耗：39.7MB

Leetcode-26.删除排序数组中的重复项(Remove Duplicates from Sorted Array)

Sep 15, 2019

采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

26.删除排序数组中的重复项(Remove Duplicates from Sorted Array)

这个题特别像之前做的27题，很简单的，用的是同一种方法即两个指针

class Solution {
    public int removeDuplicates(int[] nums) {
        int i = 0;
        if (nums.length == 0) {
            return 0;
        }

        for (int j = 0; j < nums.length; j++) {
            if (j == nums.length - 1) {
                nums[i] = nums[j];
            } else if (nums[j] != nums[j+1]) {
                nums[i] = nums[j];
                i++;
            }
        }
        return i+1;
    }
}

执行用时：2ms 内存消耗：43.4MB

Leetcode-155.最小栈(Min Stack)

Sep 14, 2019

采用语言为Java

  • LeetCode国内官网：https://leetcode-cn.com

  • 因为题号和题名都有，题目描述就不写了

简单难度

155.最小栈(Min Stack)

解法一

题目要求设计一个栈，于是开始造轮子，定义数组作为栈，其他部分按照题目要求一点一点写就好，注意要判断是否溢出，这也是很麻烦的一点，因为数组长度定义不好的话，提交时只要测试用例够长就会溢出

import java.util.EmptyStackException;

class MinStack {
    private int[] array;
    private int top;
    private final static int size = 10000;
    /** initialize your data structure here. */
    public MinStack() {
        array = new int[size];
        top = -1;
    }

    public void push(int x) {
        if (top == size - 1) {
            throw new StackOverflowError();
        } else {
            array[++top] = x;
        }
    }

    public void pop() {
        if (top == -1) {
            throw new EmptyStackException();
        }
        array[top] = array[top+1];
        top--;
    }

    public int top() {
        if (top == -1) {
            throw new EmptyStackException();
        }
        return array[top];
    }

    public int getMin() {
        if (top == -1) {
            throw new EmptyStackException();
        }
        int min = array[0];
        for (int i = 1; i <= top; i++) {
            if (array[i] < min)
                min = array[i];
        }
        return min;
    }
}

执行用时：132ms 内存消耗：47.3MB

解法二

我们人为定义一个辅助的栈box，用来存栈中最小的元素,存放数据的栈data也是利用java.util.Stack包里现成的

import java.util.Stack;
import java.util.EmptyStackException;

class MinStack {
    private Stack<Integer> data;
    private Stack<Integer> box;

    /** initialize your data structure here. */
    public MinStack() {
        data = new Stack<>();
        box = new Stack<>();
    }

    public void push(int x) {
        data.add(x);
        if (box.isEmpty() || box.peek() >= x) {
            box.add(x);
        } else {
            box.add(box.peek());
        }
    }

    public void pop() {
        if (!data.isEmpty()) {
            data.pop();
            box.pop();
        }
    }

    public int top() {
        if (!data.isEmpty()) {
            return data.peek();
        }
        throw new StackOverflowError();
    }

    public int getMin() {
        if (!data.isEmpty()) {
            return box.peek();
        }
        throw new StackOverflowError();
    }
}

执行用时：101ms 内存消耗：44.6MB
